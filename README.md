## Setting Up
1. Clone the Repo
	`gh repo clone j-a-c-k-goes/metamask-test`
2. Install Dependencies (listed in package.json)
	`yarn install`
3. Start Server (in separate termninal window)
	`yarn run start`
4. Launch App in Browser
	`open http://localhost:3000`
